Assignment 12 - SQL Exercises and HTML Application

This page includes example SQL statements from W3Schools and applies foundational HTML skills.

SQL Activity Examples

1. SQL SELECT Statement

  • SELECT * FROM Customers;

2. SQL WHERE Clause (filtered to USA)

  • SELECT * FROM Customers WHERE Country = 'USA';

3. SQL COUNT Function

  • SELECT COUNT(CustomerID) FROM Customers;

4. SQL INSERT INTO Statement

  • INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country)
    VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

5. SQL UPDATE Statement

  • UPDATE Customers SET ContactName = 'Alfred Schmidt', City = 'Frankfurt'
    WHERE CustomerID = 1;

Learning Summary

Completing this assignment helped me understand how SQL statements interact with databases. I learned how to use SELECT, WHERE, COUNT, INSERT INTO, and UPDATE commands effectively. Executing them on W3Schools reinforced my confidence in querying and modifying data. Additionally, writing and organizing HTML strengthened my knowledge of semantic tags and page layout. The inclusion of an external CSS file showed how styling can be separated from structure, which improves maintainability. These skills are directly applicable in both web development and data management environments, and I feel more prepared to build dynamic, data-driven websites.
